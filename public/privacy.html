Kebijakan Privasi

Terakhir diperbarui: 15 April 2026

Aplikasi Buwuhan Tracker ("kami") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengelola data Anda saat menggunakan aplikasi kami.

1. Informasi yang Kami Kumpulkan

Buwuhan Tracker adalah aplikasi sisi-klien (client-side) murni. Kami tidak memiliki server pusat dan tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda ke server kami sendiri.

  • Data Lokal: Semua data sumbangan yang Anda masukkan disimpan secara lokal di browser Anda.
  • Data Cloud: Jika Anda mengaktifkan sinkronisasi Google Drive, aplikasi akan mengakses folder khusus di akun Google Drive pribadi Anda untuk menyimpan file database aplikasi.

2. Penggunaan Google User Data

Aplikasi ini menggunakan cakupan API Google Drive (khususnya drive.file) untuk tujuan berikut:

  • Membuat file database di Google Drive Anda agar data dapat dibuka di perangkat lain.
  • Membaca file database yang telah Anda buat sebelumnya untuk tujuan sinkronisasi.

Kami TIDAK membaca file lain di Drive Anda, TIDAK membagi data Anda dengan pihak ketiga, dan TIDAK menggunakannya untuk tujuan periklanan.

3. Keamanan Data

Karena data disimpan di browser Anda dan akun Google Drive pribadi Anda, keamanan data sepenuhnya bergantung pada keamanan akun Google Anda dan akses fisik ke perangkat Anda.

4. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diumumkan dengan memperbarui tanggal di bagian atas halaman ini.

5. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan ini, Anda dapat menghubungi pengembang melalui repositori GitHub resmi kami.
